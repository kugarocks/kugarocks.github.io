背景#

2019-2020，當時用 PHP 寫嘅呢個活動模板項目，雖然已經停止開發咗，但而家仲喺運行，喺上面跑嘅活動已經有 400 多個咗。 我記得我離職嘅時候先 50 多個，而家 PHP 喺國內已經唔係咁流行咗，要唔係因為遷移成本大，早就改成 Java 咗。 不過得知自己寫嘅代碼仲喺運行，仲上線咗咁多活動，滿打滿算都省咗唔少開發成本，心裏仲係幾開心嘅。

今次打算把活動模板再次搭建起嚟，亦記錄一下當中碰到嘅問題。 畢竟對於一個 5 年前嘅老項目，好多依賴嘅軟件都更新咗，會出現各種各樣嘅問題。
