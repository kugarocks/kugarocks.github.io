標準文件描述符#

Standard File Descriptors。

名稱編號說明
STDIN0標準輸入
STDOUT1標準輸出
STDERR2標準錯誤

標準輸入#

命令 cat 默認從標準輸入讀取內容，如果直接運行 cat 而唔指定文件名，佢會等用戶輸入。

重定向使用 <。

標準輸出#

默認係終端或者屏幕。

重定向使用 > 或 1>。

標準錯誤#

默認係終端或者屏幕。

重定向使用 2>。

使用 &> 可以同時重定向標準輸出同錯誤輸出。

EXEC 命令#

喺命令行中使用#

Shell 進程會被新命令嘅進程取代，執行完後唔會返回到原來嘅 Shell 中。

喺腳本中使用#

腳本進程會被 date 命令取代，echo 唔會執行。

文件描述符重定向#

用於修改文件描述符時，唔會替換當前進程，只會影響後續命令嘅輸入輸出。

關閉文件描述符#

重定向到 &- 即可關閉。關閉之後，唔可以再寫數據。

/dev/null#

空設備，位桶。

把標準輸出同標準錯誤重定向到空設備。

MKTEMP 命令#

根據文件名模板，創建臨時文件。

命令會把 X 代替為隨機字符，模板最少為 6 個 X。

創建文件#

該命令創建嘅文件只有 Owner 有權限。

創建目錄#

使用 /tmp 目錄#

使用 -t 選項會喺 /tmp 目錄創建文件。

TEE 命令#

同時把數據重定向到標準輸出同文件。

屏幕同文件都有同一份數據。
