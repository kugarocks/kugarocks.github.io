LS#

文件詳細信息#

  • 文件類型：d 目錄，- 文件，l 鏈接，c 字符設備，b 塊設備。
  • 文件權限：700，Owner: 7，Group: 0, Other: 0
  • 硬鏈接數：9
  • 文件屬主：root
  • 文件組別：root
  • 文件大小：4.0K
  • 修改時間：Aug 7 22:38
  • 文件名稱：root
  • 主設備號：Major Number，5/252，標識設備驅動程序
  • 次設備號：Minor Number，0，標識同一類設備中嘅某個設備實例

  • @：符號鏈接
  • |：命名管道
  • =：套接字
