## Requirements

* Node.js - `v20.11.0` or higher — run `node -v` to check.
* Hugo extended - `v0.125.0` or higher — run `hugo version` to check.

## Setup

```bash
npm install
```

```bash
npm run dev
```

Web Server is available at `http://localhost:1313/`.
