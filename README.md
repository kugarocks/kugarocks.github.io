## Requirements

| Dependency | Version |
| --- | --- |
| Node.js | `v20.11.0` |
| pnpm | `10.10.0` |
| VitePress | `2.0.0-alpha.5` |

## Setup

Install dependencies.

```bash
pnpm www:install
```

Run dev.

```bash
pnpm www:dev
```

Checkout `http://localhost:5173/`.
